std::string WebUtils::extractTitleFromHtml ( const std::string & htmlBody)
static

Витягує заголовок сторінки (<title>) з HTML-коду.

Використовує регулярні вирази для пошуку тегу title.

Аргументи
htmlBodyРядок, що містить повний HTML код сторінки.

Знайдений заголовок або порожній рядок, якщо тег не знайдено.
