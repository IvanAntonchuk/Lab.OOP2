Структура LinkData

Структура для зберігання даних про одне посилання. Детальніше...

#include <linkmanager.h>

Діаграма зв'язків класу LinkData:

Загальнодоступні атрибути

std::string name
 Назва посилання (заголовок).
std::string url
 URL-адреса ресурсу.
std::string folder
 Категорія (папка), до якої належить посилання.
std::vector< std::string > contexts
 Список контекстних тегів (наприклад, "Робота", "Навчання").
std::string relatedUrl
 Пов'язане посилання (якщо є).
std::string comment
 Коментар користувача.
std::string iconData
 Дані іконки сайту у форматі Base64 (PNG).
